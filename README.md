# Julian Begnor — Portfolio

A single, self-contained portfolio website. Sport & Health student, future lifestyle
coach, based in Amsterdam.

## What's in this folder

Just one file: `index.html`. That's the entire website — all photos (portrait, the
gear shots in the hero, the project thumbnails) are embedded directly inside it, so
there's nothing else to keep track of and no folder structure to preserve.

## Viewing it

Double-click `index.html` — it opens directly in any browser. No install, no server,
no internet connection needed to view it locally.

## Putting it online

The site is ready to deploy as-is, no build step required. Two easy free options:

**Netlify Drop (fastest)**
1. Go to https://app.netlify.com/drop
2. Drag `index.html` onto the page
3. It's live in seconds at a random `*.netlify.app` URL — claim an account afterward
   if you want to keep it and customize the URL

**GitHub Pages**
1. Create a free GitHub account and a new repository
2. Upload `index.html` to it (GitHub's web UI supports drag-and-drop, no command line
   needed)
3. In the repo's Settings → Pages, set the source to your main branch
4. Your site goes live at `yourusername.github.io/reponame`

## Editing later

The whole site — content, styling, and the small amount of interaction JavaScript —
lives in that one `index.html` file, in plain HTML/CSS/JavaScript (no framework,
no build tools). Open it in any text editor to make changes.
